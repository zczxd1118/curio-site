2. Claude Code 推出「超码」Ultracode：操控100个Agent并行开发，解决大型任务跑不完的痛点

[AI] · ⭐⭐⭐⭐⭐ · 技术胖

Anthropic 在 Claude Code 中上线了 Ultracode 功能，允许用户同时调度最多100个Agent并行执行任务。这解决了Claude Code此前一次性跑不完大型项目的核心痛点——现在你可以拆解任务、并行开发、自动协调。技术胖的保姆级教程展示了从安装到实战的完整流程，包含具体命令和配置。

✅ 已确认 ⚖️ 尚属判断
Ultracode 是 Claude Code 的新功能，支持最多100个Agent并行工作 100个Agent并行是否会产生协调开销和token浪费，需实测验证
功能于2026年6月4日上线，技术胖在6月5日发布实战教程 该功能是否仅限Claude Code Pro用户，定价策略尚未明确
教程包含具体安装步骤、任务拆分策略、Agent协调配置 与GitHub Copilot App、Cursor等竞品的多Agent能力对比尚缺
Ultracode 解决了 Claude Code 此前大型任务超时或上下文溢出问题 实际工程中能否稳定运行100个Agent，取决于任务类型和API稳定性

这对你的 content-curator 项目是直接利好——可以用 Ultracode 并行爬取、清洗、分类多个信息源，大幅缩短开发周期。建议今晚就跟着技术胖的视频搭环境，从拆分你的Agent任务开始试水。注意控制并行度，先跑10个Agent测试稳定性。
