3. Github Copilot 改 token 计费 + Gemini CLI 停服——大厂在收紧 vibe coding 工具的钱包

[大厂 AI 动态] · ⭐⭐⭐⭐⭐ · TechCrunch

Github Copilot 把订阅制改成 token 计费，开发者社区一片骂声；Google 同时官宣 Gemini CLI 6 月 18 日停服，迁到付费 SDK。两件事拼在一起就是一个信号：大厂在烧完用户量之后，开始要求每一次调用都付钱。Vibe coding 黄金时代的'廉价 token'红利期已结束。

✅ 已确认 ⚖️ 尚属判断
Github Copilot 新版按 token 计费，订阅用户额度被显著收紧 Cursor / Windsurf 会不会跟进 token 计费，还是反向打'订阅制更省钱'广告抢用户
Gemini CLI 将在 2026-06-18 停止服务，迁至需 API key 的付费方案 对个人开发者的实际月成本影响——以你 daily 用量算大概多花多少
Microsoft Copilot Cowork 被独立安全研究披露存在文件外泄漏洞 免费 / 开源替代（Continue.dev、Aider + 本地模型）的可用性是否能补位
VS Code 一度自动给 commit 打 'Co-Authored-by Copilot' tag（无论是否使用） 企业版 Copilot 是否被强迫升级
Anthropic 与 OpenAI 的 token 价格本季度均有微涨

📖 主编点评

你做 content-curator 走 GitHub Action + Claude Code 的路线现在还安全（不依赖 Copilot），但要警觉：未来 6 个月 vibe coding 的边际成本会从'几乎为零'变成'有感知'。建议本周做两件事——1) 把高频调用的 prompt 模板化，减少无效调用；2) 评估一遍 local 推理（DeepSeek R1 / Qwen3）在你的工作流里能替掉多少。

5. AlphaEvolve + OpenAI 解几何猜想：AI 跨过'做科研'门槛的两个独立证据

[大厂 AI 动态] · ⭐⭐⭐⭐ · Google DeepMind

本月独立出现两个'AI 做科研出原创成果'的实证：DeepMind 的 AlphaEvolve 在数学、芯片设计、调度算法多领域产生人类未发现的解；OpenAI 一个未公开模型证伪了离散几何中一个长期猜想（OpenAI 官方页）。这是 LLM 从'综合材料'迈向'生成原创洞见'的拐点信号，比任何 benchmark 数字都更有意义。

✅ 已确认 ⚖️ 尚属判断
DeepMind 公布 AlphaEvolve 已产出可验证的原创算法/电路设计 OpenAI 那个模型是 GPT-5.5 / Claude Opus 同代还是独立科研专用模型
OpenAI 内部模型在离散几何领域反驳了一个公开未解猜想（OpenAI 官方） AlphaEvolve 公开 weights 或 API 的可能性（目前是闭源 + 内部用）
两个成果均经独立学术圈验证，不是公司单方面 PR 稿 AI 在'生命科学/材料'等不易形式化的领域复制此能力的速度
Demis Hassabis 公开表示 AlphaEvolve 已用在 Google 自家数据中心调度优化 学术圈对'AI 一作'论文的署名规范何时形成共识
结果均未涉及'通用 AGI'宣称，限定在可形式化验证的窄域

📖 主编点评

实操含义：如果你做电子信息工程毕设/论文，现在就该把 AlphaEvolve、AlphaProof、AlphaFold 系列做的工作流学一遍——不是用它们，而是把'AI 提候选 + 人验证'这种科研范式纳入你的方法论。下半年招聘市场上'懂 AI for Science 流程'的简历比'会写 prompt'值钱十倍。
